# Статика и медиа для сайта

Сюда складывайте файлы, которые должны быть доступны по URL.

## Папки

- **`images/`** — фотографии, картинки, иконки для сайта.

Файлы из `public` отдаются с корня сайта. Например:
- `public/images/hero.jpg` → **/images/hero.jpg**
- `public/logo.png` → **/logo.png**

В коде используйте путь с ведущим слэшем: `/images/photo.jpg`.
